第一章

一道闪电划亮了漆黑的长空，落地窗后的人望着几个月以来难得的倾盆大雨，嘴边牵起一丝微笑，令他魅惑众生的容貌更添亮彩。可惜他身边的男子却不为所动，
大手举起一杯酒，笑道"怎么，又有哪个不怕死的惹到你啦？"
窗边男子亦拿起一杯酒，其优雅的动作既使是受过专门训练的调酒师也要自愧不如。
"没有，只是想起前些日子伟说的话。"
"伟说什么话，竟然可以使冷血的你笑起来？"
轻含了一口酒，仔细品赏了一下才说"伟说我在这个世界没有缘分，要到另一个世界寻找，不过我和那个有缘人却没有名分。"
"什么怪论调，另一个世界，伟这个占卜师似乎想是混口饭吃而已，不必当真。"
"不，伟不是那种信口开河的人，而且这个世界我也玩厌了，换一个地方玩也不错。"


**** **** **** ****


一栋雄伟的建筑中，人人像繁忙的蜜蜂一样，忙个不停。此时 ，自动门打开，一个俊得不像话的男人走了进来。不是时下一般少女迷恋的清俊模样，也不是前些日子流行的酷男，他的五官立体，双眼不是虎目，而是连女人都恨不得拥有的凤眼，微微一眯，散发出万伏电流，连男人也难挡其万一。希腊式的挺直鼻形，诱人犯罪的艳丽薄唇，看上去有点单薄的身子却有着1.82米的身高，完美的九头身比例，标准的衣架子身材，无论他穿什么衣服都可以穿出其独有的品味。
但忙碌的人们见到此美男却不见任何迷醉的目光，反而个个像见到的阎王一样怕得连冷汗也快出来了。对，他正是商界上人人惧称的"玉面阎王"--龙风。龙风能在商界上呼风唤雨不是偶然，他强硬的作风，对经济的绝佳触感令他在商界上无往不利，但他也像是没有心似的，只要有利益，便不顾人死活，往往令人家破人亡。所以，商界上一谈起他都谈虎色变似的。可是他的绝美容颜也令一群不怕死的狂风浪蝶为他疯狂不已。
无视身边人们一幅惊怕的表情，龙风心情愉快地走进他的专用电梯，因为伟预言的日子差不多到期了，他挺期待他会到什么世界去。突然一阵摇晃，电梯的电灯一闪一闪，瞬间便漆黑一片，再出现光线时，龙风发现自己已身处另一世界里了。


**** **** **** ****


唐朝开元盛世，作风相当开明，不管你是何种种族的人，都能在长安的大街小巷找得到，妇人也能随意在街上行走，更不用说她们的衣裳也是数个朝代中最为"轻凉"的。
龙风到这个时代也已经一个月了，随意的四处游玩，他从各地的富公子身上拿到的财物也差不多用完了。看着自己身上的雪白衣袍似乎有变灰的趋势，他走进了一家有名的酒楼。普一走进大门，一声声抽气声不绝于耳，一道道惊艳的眼光向他投射过来。龙风不为所动，要了一间上房和一些酒菜便随小二上楼。店小二几乎是一个口令一个动作，好不容易才回过神来，为这位娇客布置好一切。
换过雪白的衣袍，有些长的柔软黑发用一条雪绸带随意束起，吃着酒楼的招牌菜，心想：这个伟也真是的，说什么另一个世界，让我有些期待，谁知只是回到了古代而已，也没有什么好玩的。
"啊！"一声尖叫声划空传来。
龙风被这声尖叫打断了思绪。
紧接着碗碟摔破的声音不断传来，似乎下面正在撕杀一般。
龙风皱了一下眉，他最讨厌别人在他想东西的时候传出杂音。


**** **** **** ****


放下了筷子，龙风走出了房门，探头往下一看，只见楼下不知何时，来了两女一男，两位女子生得是花容月貌，一位身穿月牙色的衣裙，头带金钗，另一位以花为钗，身穿粉红色的衣裙，倒是男子正和别人交手，看不到样子，但以他的身手来说，应是数一数二的高手，因为只用了几招，和他交手了人已躺在地上扒不起来了。
像是有所感应似的，两位姑娘往上一看，都不由倒抽一口气，什么时候看过这样优雅的人物，那长相简直令她们妒忌不已，但在情窦初开的心灵里，那个如天仙一般的人物，已占据心灵，不能自己了。
龙风看了一会，见没什么好玩的，就要转身回房间。
"公子，可是要用膳，不如和奴家一起用膳，补偿打扰公子用膳的罪过。"两声娇唤同时响起。
龙风冷哼一声，转身就要走开。
"两位姐姐好意邀请你用膳，你居然一声不响就走开，好大的胆子。"不友善的声音从身后响起。
龙风转头一看，不知何时，那名男子已从楼下来到楼上。龙风冷笑一下，已经好久没有听过这么没礼貌的说话了，竟有点怀念起来了。
"怎么，见我没有答话，便替美人出头了，要知道你的大美人可是在钓我啊！想不到来到唐朝，女人也是一样的烦。"
虽不清楚龙风在说些什么，但任谁看见龙风的模样也知道这不是什么好说话。
"你不要以为你俊得不像话，我就不敢打你，要知道谁敢惹我姐姐不高兴，我就不放过谁。"
龙风认真看了这名男子一下，只见他龙眉虎目，小麦色的皮肤上有些细小的伤痕，穿着一身短褂功夫装，年纪约有十八、九岁左右，谁比不上他两位姐姐的绝丽容貌，但也算得上是一位充满男子气慨的美男子了。腰间佩了一把宝剑，但从他刚才不用剑便能打倒几个剽形大汉来说，他武功应该十分了得。
两个美男子一站起来，龙风比起这名男子还要高一些，这也是男子不高兴的地方之一，为什么这个娘娘腔的家伙比他还要高。
龙风微微一笑，让人看不出他想干什么，只是一个劲的沉醉在他的微笑中。"这么吓人啊！那我只好陪你们吃顿饭了。"


**** **** **** ****


大家分位置坐好，两位姑娘坐在龙风的左右两边，男子只好 坐在龙风的对面。酒菜陆续上来，龙风难得十分有礼貌地询问两位姑娘的闺名。
一位姑娘羞答答地回答道："奴家姓钟，名为真。"又指着对面的粉红色衣裙的姑娘道："她是我的妹子，名晴。这位是我的三弟，名实。"
龙风笑了，"你爹还真有文化，居然帮你们取名叫忠贞、钟情、忠实。难不成还有忠心、忠诚、忠厚。"
"公子，真是好聪明，我家虽没有忠厚，但我大哥、二哥就是叫钟新、钟诚。"
"姐，你和他说什么，快点吃饭吧，我都快饿坏了。"
"真没礼貌，公子，还未请问你贵姓了。"
龙风望着她故作矜持的样子，心中冷笑一声，"我爹没有你爹那么有才华，只给
我取了一个风字，你可以叫我龙风。"
"那我唤你龙大哥可好？"
"你还是唤我龙公子好了，免得你弟弟说我损坏你的名誉。"
钟真有点忧怨地看了龙风一眼，见他连看都不看自己，埋怨龙风的不懂风情，有点生气地端起饭吃。
钟晴望见姐姐被龙风拒绝，有点高兴地笑了笑。整顿饭就在这种气氛下结束了。为了可以再看心中人一眼，钟真姐弟三人也同样寄宿在这间酒楼的上房中。谁知第二天
两姐妹再去找龙风的时候，龙风已经走了。

玉面阎王 第二章


龙风走在长安最旺的一条街上，手上拿着刚"借"回的那些富家公子的钱财，心中不禁为昨天自己居然和自己最讨厌的女人同桌吃饭感到迷惑，可能是想看一下那个钟实发怒的样子吧。因为这个钟实还真的对他的两个姐姐有恋姐情结。
不过这只是一时的乐趣而已，过后却没有什么成就感，真不知道什么时候才可以回去，起码未来还有几个不怕死的兄弟陪他玩。
不知不觉龙风已走出城门，来到城外的森林中了。忽然听到一阵兵器撞击声，不怕死的过去一看。
"哦？"
龙风不禁吃了一惊，昨天那两个文弱女子钟什么的，现在居然拿着剑，和那个钟实一起凶神恶刹地对付一个受重伤的男子。那男子长发披肩，远远地看不清模样，浑身都是鲜血，一件灰色的衣袍都快变成血色了。
"你们还是不肯放过我吗？我娘已经为了我已经自尽在你们堂前了。我们欠你们的已经还清了。"
"哼，狐狸精的儿子，我告诉你，我娘亲对你那低贱的娘是永远都不会原谅的。只有你们全都死了，我娘才解心头之恨。而且你已是人不人，鬼不鬼了，何不和你的娘一起到地狱去呢？"钟真一脸狠毒地说。
"本是同根生，相煎何太急，虽不是同母所生，你.....你....我也是同一个爹啊！"男子痛苦地说，一口血喷了出来。
钟真闪过血，"我才没有像鬼的兄弟呢！还有不要乱叫，我爹才不是你爹呢!"
再一剑剌出，男子大喷鲜血，像是死了一般倒向地上。
钟真三人探了探他的鼻息，才抹干剑上的鲜血，踢了一下地上的死人才走。
"真是最毒妇人心，看来古代的女人要比现代的女人狠。"一边走向地上的死尸，一边自言自语地说，龙风查看了那男子一下，发现仍有一丝很微弱的气息，"就算是捡一个玩具回去吧。"龙风抱起那名男子，往森林深处走去。


**** **** **** ****


虚弱地睁开眼睛，幻世望了望陌生的房间，满屋的药香扑鼻而来，屋子不大，除了自己的睡的木床而外，只有一张木桌和两张椅子，窗外隐隐传来鸟鸣声，阳光透过虚掩的门射过来，十分柔和。
"我还没有死吗？"疑惑地望着身上的绷带，幻世挣扎着爬起来。
"你在干什么，伤都没有好，就想再伤一次？我可没那么多闲功夫照顾你。"一个冷漠地声音传来。
沿着声音的源头看去，阳光下似乎站着一位仙子，黑发半披，雪白的脸孔上带有一丝倦色，但丝毫不损她的美丽，幻世有些痴了地看着仙子。忽然想起自己鬼一般的脸，立刻掉过头去，不敢折损仙子的眼睛。
"还藏什么，要看我早就看过了。"磁性的声音不悦地传来。仙子的声音怎么像男子一般，幻世有些奇怪。
"还有，我是男的，不要想些无谓的事情。"龙风从他的眼中看出他的疑惑，难得好心地为他解说。
"你是男的。"像是松了口气，又像是有点遗憾地口吻。
龙风不再理他，只是拿起药给他喝。"喝完它，想我活了二十五年，还是第一次帮人熬药。快点喝完它。"
幻世不再言语，静静地喝完药，慢慢地又沉睡起来 。


再次醒来，已是日落时分，桌上点着腊烛，龙风正在吃饭，"请问恩公尊姓大名，幻世一定会报答恩公的。"
龙风一脸冷静，并没有被突然出声的幻世吓到。"你叫幻世，很奇特的名字。我叫龙风，你不用恩公恩公的叫我，我救你可是有目的的。"
"幻世已是无用之人，我想可能帮不了恩公。"
"叫你不要乱叫，你可以叫我龙风，不过我告诉你，我救你，只不过是我刚好缺了一个仆人，又看你挺顺眼，才救你的。你不用感激我。"
"只要公子愿意，幻世愿意用一生偿还公子的救命之恩。"幻世一口答应。自小就孤苦一人，唯一的娘已经过世，还有什么值得在意呢。
"好，等你伤好了，你就是我的仆人。"


**** **** **** ****


随意地在街上走着，龙风不理会别人投射过来的惊艳眼光，反正天生美貌难自弃，如果有哪个呆子来提供一点乐趣更好。
回到客栈，幻世已经整理好一切行李，龙风结帐以后，两人走在大街上，和刚才的惊艳眼光不同，这下子眼光全变成了惊恐了，幻世垂着头，一点也不敢向前看。龙风看着途人和刚才不同的反应，心情愉快地向前走，还从未被人当怪物看过呢。
在前往洛阳的途中，龙风仔细地看着幻世，说实在的，幻世其实并不丑，五官像刀刻一般深刻，身材强壮有力，和他的身高比起来，两人好像也差不了多少，搞不好，他还矮一点。两人比起来，幻世混身充满男子汉气概，和自己中性的美貌不同，相信是很多女子心中的良人。可惜的是，他的脸上布满了伤痕，有刀伤，也有烫伤的痕迹，一张好好的脸就这样毁掉了。而且他的自尊心好像也毁得差不多了。一个大男人低声下气地服待自己，一点也没有心生不愤。
对龙风充满打量的眼光，幻世尽量装着视若无睹，自己的丑自己知道，其实他真的不配站在天人一般的龙风的旁边，但龙风好像真的一点都不在意自己的丑陋。对龙风他真的是当主子一般看待。从小所受到的厌恶、憎恨、惧怕的遭遇，让他知道龙风也会有嫌弃他的一天，在这天还未来临之前，他一定要好好侍候龙风

 

第 三 章

一路相安无事，龙风来到了唐朝第二个城市--洛阳，龙风决定买间房子住下来好避避暑。
‘真是的，古代的人真是老古董，天气这么热，还穿这么多衣服。"龙风一面扇扇子一面嘀咕着。看着正在砍柴的幻世，一点汗也没有，难道他不是人吗？龙风有点怀疑。
幸好救了他，现在有个人服待他，衣服脏了有人洗，肚子饿了有人煮饭，茶没了有人冲，在这么热的天气里，什么都不用做，还真是惬意啊！
看着幻世一身古铜色的肌肤，说实在的，龙风还真有点羡慕，望着自己总是雪白的皮肤，叹了一口气。不知不觉睡着了。
幻世把柴抱进厨房后，来到大厅就看见龙风躺在长椅上睡着了。看着龙风雪白的脸庞，无瑕的肌肤，幻世看呆了。明知他是男子，却总是被他的绝色的脸所迷惑，平时带着冷漠眼光的眼睛闭起来，显出了长长的睫毛，柔软的长发不规则地散在额头上，艳丽的薄唇像是邀请似的微微张开，轻轻的伸出手，轻抚着雪白的肌肤，从额头到面，绕着美丽的唇线轻轻地画着。两人的气息越来越近，就在两唇贴近的时候，龙风的一声轻呓打断了一切。
我在干什么？幻世迅速地弹开，双膝跪下，两手抱着头，痛苦地想着。他是一名男子，他是你的恩人，你想对他干什么，你竟然想用污秽来沾染他，你只是一个仆人，你只是一个丑陋的人，你凭什么抚摸他。拼命地打着自己，嘴角已经流出血了，仍不知觉。被拍打的声音吵醒，龙风睁开双眼，只看到幻世不断地打自己。
"你疯了？住手，你干什么打自己？"捉住幻世的双手，龙风问道。
像是碰到脏东西似的，幻世迅速甩开龙风的手，站起来， "没什么！你饿了是 不是？我去煮饭。"说完，不理龙风的反应，立刻向厨房走去。
"搞什么鬼嘛？"龙风摸了一下头，不解地骂着。


**** **** ****

幻世闪开她的剑招，钟新快速地制住钟真，"你疯了，他是你弟弟，你为了一个外人要杀你弟弟。"
"他才不是我弟弟，我要杀了这个贱民，他不配当我们钟家的人。"


**** **** ****


"啪"的一声，钟新重重一巴掌打过去，钟真呆了，一直对自己宠爱有加的哥哥，今天竟然打她。
"来人，把小姐带走，没有我的允许，谁也不能放开她出门，另外告诉城东的李家，我答应李家的求婚，叫他们找个吉日上门提亲。"钟新唤来家丁，把钟真交给家丁押走。


**** **** ****


"小弟，我一直都没有好好照顾你，年少无知的我毁了你的容颜，你不会恨我吧。"钟新握着幻世双手羞愧地说。
"不会了。大哥。"幻世摇摇头，
"如果他是你认定的幸福，你一定要捉紧它，不要让它飞走。我知道我没有资格说这些话。但我是真心诚意地向你忏悔，希望你能原谅我。钟家永远是你的家，不管你受到什么委屈，都可以回来，哥一定会照顾你的。"
"龙风，我把小弟交给你了，你一定要好好对待他。"
"不用你说，我也知道。"龙风不高兴地甩开钟新握住幻世的双手。说话就说话，干嘛还要握手。
"谢谢你，哥，虽然我恨过你们，但你永远都是我最敬爱的哥哥。"幻世有点哽咽地说。
钟诚也不甘落后，给了幻世一个大拥抱，当然又被龙风推开。"我也一样，我衷心祝福你，如果你受委屈，二哥一样可以给你靠。"


**** **** ****


挥手告别送行的钟家人，龙风和幻世再次踏上去长安的大路。和来时不同，幻世和龙风怀着全新的心情，走向回未来的路途。

 

尾 声
一年后。
顺利地来到未来，幻世努力地学习着未来的一切，凭借着天生的聪惠和后天的努力，幻世基本上已经适应未来的生活。
靠在办公椅上，现在幻世是龙氏企业的行政副理，除总裁龙风外，公司就属他最大。想当初一到未来，也是落在龙风的办公室内。刚恢复视线就看到五个男子或坐或站地围着龙风和他。
后来才知道，这五个男子和龙风是同一间孤儿园出来的，六人情同手足，共同创下一片天。现在只剩下龙风守着龙氏，其他人各自寻找自已的乐趣。其中以龙伟的乐趣最怪，他凭借天生的预感，立志做一个最伟大的占卜师。
今天龙伟说了几句话，让他不由停下手中的活发呆起来。
龙风一进幻世的办公室，就看见他发呆，关心地探了一下幻世的头，还好没有病。但幻世在发什么呆。
幻世回过神，看见龙风美丽的容颜靠近自己，不由一阵脸红。推开龙风，龙风看风桌面上摆着几盒精美的点心。
"是谁拿来的，我还想找你一起吃饭。"
"是楼下的女职员送给我的。我只好收到下了。"
凭借着现代出色的整容手术，幻世脸上的烫伤已被磨平，剑伤也只剩下浅浅的疤痕而已。结果，恢复八成英俊的幻世再加上他的好脾气，成为了公司最受女性倾慕的男士。龙风也为此常常吃醋。
想起龙风对这件事的反应，在龙风还没有 发火前，幻世连忙转移他的注意力。
"伟今天说了些话，他说以我和你的外形来看，你应该是受，而我是攻，还问我你的滋味如何。说实在的，你也应该让我做一次吧。"
龙风脸色大变，干笑几声，"我有点事，先走了。你慢慢吃点心好了。"
一面快速退出办公室，一面咬牙切齿地想着，伟你该死，竟然提醒幻世这件事，我非要煎了你的皮不可。


**** **** ****


正在研究命理的龙伟感到一阵寒意，预感自己大难临头，立刻收拾行李，还是到龙焱那避一下吧。

（完）
