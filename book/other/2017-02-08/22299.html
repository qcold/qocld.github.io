东城凤一边用火焰阻挡着蟒蛇的胆汁，一边用了风之舞者的力量刮起了沙石，沙石全部朝着蟒蛇移动，随后又用水元素的力量把干燥的沙石拌成沙浆，用沙浆把蟒蛇禁锢住。

东城凤将朱雀圣火的力量集中在剑气中，朝着蟒蛇的头部一刀劈下。

然而蟒蛇的头部硬如铁，东城凤的这一刀仅仅是砍了蟒蛇的半个头，疼痛使蟒蛇疯狂了起来，原本禁锢住它身体的沙浆被闪开。疼痛难忍的蟒蛇开始狠狠的撞向东城凤。

“小金，用点力气。”东城凤第一次讨厌自己的力量不够，早知道他以前要勤劳的修炼了，今天也不会被一条蛇给欺负。

嚎………………

小金一边龙吟一边把自已的力量贯彻到东城凤的身上，但是它到底是一条被封印过的龙，力量还是有限。

而凭着这一点力度，却是怎么也砍不倒蟒蛇。

蟒蛇推着东城凤向前冲，而他们的背后就是东翱的城墙，这样一来不但自己和小金有生命危险一旦城墙倒了，那里的百姓一定会被蟒蛇袭击。

东城凤开始闭上眼晴，将自己的注意力集中起来，口中开始喃喃自语：金、木、水、火、土我亲爱的五行精灵们。请听从创造你们的天地之灵，给与我力量。

天地之灵就是天帝的灵魂。因为天帝的灵魂是由天地之灵气孕育而成，而他本身是第一任天帝的精魄，他的身上隐藏着第一任天帝的力量。

第一任天帝又是吸取天地之灵气、日月之精华而经过长年累月的磨练诞生而成的。

这也就是东城凤为什么能够运用五行元素的真理所在。

金色、青色、白色、红色、黄色五种天地之颜色结合在一起普照在东城凤的身上发出了耀眼的光芒。

啊… …

东城凤的呐喊声响遍了整个沙场，他拥进了所有的力量紧紧的握紧了手中的剑，朝着蟒蛇的身体劈了下去。

在他的身体即将撞上城墙的时候，蟒蛇被劈成了两半儿倒在了地上。

蟒蛇的赴内恶心胆汁，以及无数的小蛇倾巢而出。

东城凤无力的身体开始往下掉，千钧一发之际金龙拉住了他飞向了干净的地方，不然一定掉进了蛇堆里。

“不……”这一边魔王痛苦的喊叫了起来，黑色的力量更加的强大了，同时围着他的四个人弹了出去。

蟒蛇是魔王的朋友，唯一的、忠诚的、忠心的朋友，就像另一个他。

所以这一刻魔王疯狂了，新仇旧恨一起冲向了东城凤。

而此时的东城凤根本已经没有了反抗的力量，只是虚弱靠在金龙的背上，眼睁睁的看着强大的黑色力量向自已袭来。

第二十七章凤吟

“圣儿。”金色的神剑滑开了东城凤和魔王之间的距离，银色的雪狼飞奔而去叼住了东城凤的身体往一边避开。

因为人的动作没有狼的本体来的快，但是尽管如此狼痕还是被伤到了。

鲜红的血从他的口内喷出。染红了银色的毛发。却仍然用自已的身体挡住了东城凤，狼的忠诚就像他对伴侣的专一一样。

天色开始灰暗了起来，龙焱寒一头金色的长发几乎拖地，原本紫色的长衫已经被纯白的长衫代替。衣衫的表面是生龙活虎的九爪金龙。

金色的神剑开始叫嚣着，散发出耀眼的光芒。俊美的容颜风华绝代，而他这张容颜凡是认识的人他不会忘记，那便是尊帝……东城吟。

而另一边一身黑衣的魔王虽然如孩童的娇小。但是也没有人会将他当成一个孩童来看持，黑色通体的魔剑，同样看着神剑吆喝着。

秀美的童颜没有孩童的天真。取而代之的是深深的恨意。

“我恨你，跟恨东城凤一样的恨你。”魔王的声音低沉而苍老。不属于孩童的清醇。

“我知道。”龙焱寒慵懒的声音也不似曾经的随意，而绝对的认真。

“你不知道。”魔王的声音里有些嘲笑：“我不是恨千年来魔族与神族的瓜葛，我恨的是你的残忍，你看似无害却比任何人都冷心，我是魔、你是神，但是你比我像魔、因为你比我无情。”

“你是从古至今最有情的魔。”东城吟开口：“我也从来说过我是神。从朕登基的第一天朕就说过，朕的一生不为任何人而活，是那些人愚蠢的将白已的思想寄托在朕的身上，妄想统一天下，朕从来都不屑，但是… … 。”

“但是东城凤出现了。所以改变了你，也改变了月，你都喊着自已无辜、都装着自已可怜。但是你们谁明白、又有谁明白东城邪月的心。

他被不负责任的父亲抛弃了。把本不该在他那个年龄段承受的责仕压在他的身上，他被自己疼爱的弟弟利用了，把一份不该有的情套在他的身上。

可最后他因为那些女人的自私而爱上了自己的儿子，却到头来只是为他人作嫁衣裳，他的儿子爱上了他的父亲。他才是那个最无辜的人，而你们通通该为你们自己所做的一切付出代价。”魔王偏激的认为是所有的人负了东城邪月，可是他又可曾明白。被东城邪月曾经伤害的人又该去怨谁。

“既然如此你为什么不陪他好好的过，却还要兴风作浪。”东城吟怎么也想不明白他的思想。

“因为我要看着你们痛苦。我要让你们痛苦、痛苦。”苍老的声音呐喊了起来。手中的魔剑仿佛被注入了生命开始吸取着不同的黑暗。

轻巧的身体向着东城吟的地方攻去。东城吟握紧了手中的神剑，迎向了魔王，半空中两人的身体打斗了起来，只看见黑色的影子和金色的影子在不停的移动。

两个人的力量都到了前所未有的高点，谁也无法接近他们。

黑色和金色直冲天际。将整个大地分成了两种颜色。

砰………

两种力量发生了强烈的撞击，突然一道黑色的身影从半空中掉了下来。

噗嗤……

鲜血从魔王的口中吐了出来。

怎么会？东城吟纳闷，魔王不可能只有这些功力的。

果然，魔王朝着东城吟笑了起来，再一次的凝聚了手中的黑色力量。而这一次他袭击的不是东城吟，而是东城凤。

“不… … 。”东城吟大声的喊道。

可是来不及了魔王手中的力量已经挥向了东城凤。

东城吟的脸色顿时变的苍白。

一股温和的力量从一边飞速的传了过来。另一道娇小的身影在黑色力量袭击上东城凤之前。挡在了前面。

而身体的主人不是别人正是跟魔王长的一摸一样的一个小孩子。

魔王不敢相信的看着东城邪月的身体被撞了出去：“不…… … 月……为什么……为什么……。”

魔王痛苦的喊叫着，为什么？

东城邪月的身体撞进了东城凤的怀里，两个人又一起被撞飞了出去。

“圣儿……”东城吟和魔王一起跑了过去。

“月，为什么……为什么…？”魔王拉着东城邪月的手哭泣了起来，纵使有着神之雪莲的力量，但是东城邪月的肉体根本受不了魔王刚才的一击。

他的嘴里不停地流着鲜血，呼吸也开始变得急促了起来，只是他的手伸出抚摸的却是东城凤的小脸。

而东城凤已经傻傻的忘记了反应，只是眼睁睁的看着东城邪月，甚至还不知道此刻躺在自己怀里的人就是东城邪月。

“凤，一直以来……一直以来，我都欠你一句对不起，对不起… … 我伤害了你……对不起， 但是我爱你。”

东城邪月随后将视线转向魔王；“谢谢你，一直以来谢谢你。”

“你不是都忘记了呜？不是将过去都忘记了。为什么。为什么会想起来？”

“因为神……神之雪莲的力量，你忘记了，它能解百毒，你每天给我服用的忘情散自然起不了多少的作用，有… … 有一段时间… … 我… … 我遗忘过… … 。”东城邪月的口里又吐出了一口鲜血。

“不… … 月不要说了。我马上给你疗伤。不要说了。”魔王想阻止东城邪月。却被东城邪月拒绝了。

“让我……让我说完，跟你在……在一起的时候，我很幸福……真的……真的很幸福……，就像曾经……… 曾经跟凤在一起的时候…… 一样。但是我… … 我伤害了他……所以… … 所以不想… … 不想伤害你，还… … 还有… … 。”东城邪月的气息开始慢慢的淡了：“有下辈子… … 下辈子我… … 我陪… … 陪你… … 。”手轻轻的滑落。

东城邪月直到这一刻才永远的闭上了眼晴。

同一时刻一滴晶圣的眼泪从东城凤的眼眶里流出，他不明白他为什么会哭，只是心好痛，看到这个人躺在自己的怀里，心好痛。

“吟… … 。”东城凤呆呆的拉着东城吟的衣服。

“他是你父皇。”东城吟将东城凤抱进怀里，轻声的安慰道。

“父……父皇… … 。”父皇已经是很遥远的记忆了，但是为什么此刻想起来却是那么熟悉。

眼泪又不受控制的流了下来，一滴一滴的滴到了东城邪月的身上。

忽然，东城邪月的身上发出了七彩的霞光。

这是？

所有人都不明白的看着这一幕。

因为东城邪月曾经食用过神之雪莲，而东城凤第一任天帝的灵魂本来就是从神之雪莲的莲子里孕育而成。

所以两个人的灵魂产生了共鸣。

东城凤的眼泪参合着生命之水的力量，让东城邪月再一次的重生了。

原本在东城凤怀里断气的身体变成了足月的婴儿，婴儿的身上还挂着东城邪月的衣服，但却是一个完全灵魂。

婴儿的小手紧紧的抓着魔王的衣服唧唧喳喳的笑着。

一年后

东城凤坐在御书房里叹了第一百零八声气了。

“陛下……陛下……… 。”伊人好笑的看着无精打采的东城凤也不禁的摇头。

“怎么我在门外就听见你的的叹息声了。”龙焱寒端着一碗清淡的粥走进御书房：“无聊了？”

将椅子上的小人儿抱进怀里哄着。

“吟，作皇帝好命苦啊。”东城凤抱着龙焱寒的脖子撒娇着。

一年前，西麟因为发生了西麟太子事件之后。西麟的皇帝废了太子。改立西煜飘为太子，一个月之后西麟皇帝病逝。西煜飘登基为帝。

于此同时，东城洛亦以身体不适为由让位与六皇子东城凤。

东城凤因为上次东城洛亦差点被轮暴的事情而无法拒绝，抱着对皇位的好奇满口答应了下来。

而其实通过这件事情东城洛亦也知道自己并不适合当皇帝。

“大哥在西麟不知道会不会被欺负？”东城凤玩弄着龙焱寒胸前的衣服。低语着。

“你又想逃了？”不用猜也知道，这个小家伙自从登基之后一年里已经逃了几百次了。

“没有。”东城凤摇头抗议：“我打算以两国友好为名，光明正大的去看大哥。”

瞧瞧他为自己找了一个多少好的理由啊，东城凤有些得意的想到。

做三界之主需从人界之主开始体验。代表着天帝灵魂的魂兽还没有找到。所以离东城凤成为天帝的日子还远着呢。

而真正的且听凤吟从这一刻才刚刚上演。
