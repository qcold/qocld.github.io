他叫小欣，爸爸妈妈都是池族的人，偏偏他是江族的公主，这也就注定可他有两个名字，在江族人嘴里他叫江欣，池族

人叫他池欣，这些他都已经习惯了。

池欣十八岁厨师技校出来以后，本以为终于可以开始自己吃遍全球美食的梦想了，谁想到两族的大人开始给他张罗着相

亲了。

相亲啊，不好玩，只有相亲大餐味道还是不错的。

几十几次的相亲下来，大人问池欣。

「你感觉怎么样？」

「怎么样？上次相亲那家饭店的东坡肘子做得很好，再上次那个饭店的菊花鱼我很喜欢。」

「哈哈！」

池欣抬头，大哥池威不知道什么时候进来的。

「大哥你笑什么？笑得花枝乱颤的？」

大哥过来拧了拧池欣的脸。

「池欣，你脑子里也就只剩下吃了，你看看你把爷爷气成什么样子了？」

池欣抬头果然爷爷脸都绿了。

「我说错什么了吗？」

大哥拍了拍我的头。

「没有，你竟然相亲相到只记得起相亲宴上的菜色，也就说明对面的雄性不吸引你，说明那些雄性不出色所以也就没什

么好谈的了，爷爷您再努力吧。」

池继咬牙，池威这孩子不气得自己吐血不算完的。

「小欣啊，你到底喜欢什么样的？找雄性还是池族的好啊，最少肤色相同，你看看江族那惨白的颜色，看着多难看啊。

」

池继找了机会就贬损江族。池欣虽然入了江族籍，可池继就是不甘心想让池欣的下一代是纯池族血统。

池欣摇头，心里真的没什么概念，才出学校让他嫁人，诡异啊。

趁着爷爷出去一会儿，池欣问大哥。

「大哥我看你们刚才在收拾行李是不是要出门啊？」

「你别想了，爷爷已经决定带你一起去了，十几个青年才俊在那边等着见你呢。」

「啊！」

最后池欣还是没有跑掉，还以为爷爷和哥哥姐姐去开什么龙族老大会议没他什么事的。

他们去开会，池欣除了几次例行的相亲就是待在饭店无聊的要死，还有就是要说，这边的食物真的好难吃，中餐做得也

不专业，忍耐了一顿晚餐，第二天中午这一餐池欣实在忍无可忍，跟饭店借了厨房开始自己做饭，话说学厨师就这点好

处自己可以给自己做美食，尝到好吃的可以问了食谱自己回去做了吃，永远不亏嘴。

「冰糖少许、酱油两钱。」

把成品放到鼻子下面闻一闻，池欣的东坡肘子出锅了。

等不及回房间拿上刀叉坐在饭店的餐厅里开吃，爷爷他们会还没散，估计不会回来吃了。一桌子做了三十个菜，都是我

和保镖们的了。

把袖子向上面挽一下，开吃。

吃的眉开眼笑的池欣没有注意，离着他不远还有一个对着饭店提供的伙食愁眉不展的仁兄。

唐纳尔·埃伯斯。人形外表为壮硕男一个，龙族雄性海族人，现任龙族联合事务处安全部司令，负责本次龙族首脑会议

的安全，会场那边足足忙碌四十八小时安排妥当回来饭店睡了一觉起来，饭店的伙食让他开始为难。

眼前的东西说不上色香味，总之看着就没什么食欲，摆明告诉他是营养餐。

硬着头皮才要开始吃饭就看池欣那桌上了一桌子丰盛的午餐。

唐纳尔·埃伯斯有些不开心了。

「侍者。」

服务员匆匆跑过来。

「唐纳尔先生您有什么需要帮忙的吗？」

唐纳尔·埃伯斯指了指池欣那一桌。

「为什么他的饭食看上去那么可口，我要吃这些东西，你可以给我一个解释吗？」

服务员面有难色，吞吞吐吐的说。

「唐纳尔先生，那一桌的饭菜不是我们饭店提供的，是那桌的客人自己做的。」

「恩？他是哪一族贵族的随行厨师吗？」

「不是，他好像自己是个贵族。」

皇族的出行力求保密，所以饭店的服务员并不清楚池欣的来历。

唐纳尔·埃伯斯叹气，开始摆弄自己的手机，突然误按了波长探测器。

红色，那个吃美味大餐的人是个龙族雌性，也许有门，凭借着自己的身份还有自己的雄性魅力，唐纳尔·埃伯斯决定很

没面子的去蹭人家一顿饭，虽然被人知道会很丢人，但是总比熬夜四十八小时以后还要亏待自己的肠胃强啊，

唐纳尔·埃伯斯站起来靠近小欣那边，才到桌边，小欣的两组保镖同时站了起来。

这里说一下小欣的保镖为什么算两组？本来每一个小龙身边都有六个保镖兼助手，小欣因为地位的特殊，六个保镖江族

、池族各出了一半，三个人一组。

唐纳尔·埃伯斯愣了一下，然后耸耸肩。

「对不起打扰了，我没有恶意，只是想过来聊聊天。」

池欣抬头看看左右，是什么人，主要住这里的各族政要们会议还没有散，现在也才十点半，不是用餐时间餐厅没什么客

人，细看一下就他们这里七个人算热闹了。

池欣对手下挥挥手示意他们坐下。

那个不请自来的外国人唐纳尔·埃伯斯说，

「我方便同桌吗？」

池欣转头问助手。

「他问可不可以同桌。」

「坐吧，没人拦着你。」

池欣对紧张的保镖们示意。

「不要那么紧张，坐下吃饭。」

保镖们重新拾起筷子，话说七个大男人吃饭简直就是风残云卷，突然对面的外国人说。

「你们吃的不是饭店的营养餐吧，看上去很特别，是哪国菜？」

怪异的腔调让池欣再次从饭菜里面抬头。

「你会说我国的语言。」

「是的。」

「我做的这是中国菜，怎么，没吃过？」

池欣热情的塞了一双筷子给那个外国佬。

「尝尝看，比饭店营养餐好吃多了。」

外国佬的筷子伸缩如飞，慢慢连专注与吃饭的池欣都觉得他似乎比他们吃得还快。保镖们和池欣一样放下了筷子看着那

个狼吞虎咽的人。

那人吃了好半天才意识到人们盯着他看。抬头露出尴尬的笑。

「不好意思，味道实在太好了，尝得是不是多了一点儿。」

自己做的菜有人欣赏是件很开心的事情，池欣对那人笑了笑。

「没关系，喜欢就放开吃，明天这个时间我会做别的菜色，有兴趣一起过来吃啊。」

池欣其实也就是客气那么一说。一个陌生人他还能每天过来蹭饭，好意思啊。

「好！」

那人倒不客气。

饭碗朝天菜汤都不剩的一餐过后，池欣和那个外国佬几乎可以称作朋友了。

「我叫唐纳尔·埃伯斯。」

「我叫池欣。」

大家介绍完毕，唐纳尔·埃伯斯问池欣。

「你是过来开会吗？还是旅游？」

「我家大人来开会，我不懂，只能闷在饭店里。」

「其实这里有很多名胜，你可以出去转转。」

「人生地不熟的又没有导游。」

唐纳尔·埃伯斯想了一下自己的日程安排。

「下午我有事，我让我的助手带你们出去看看好了。」

唐纳尔·埃伯斯召来了自己的助手给池欣他们，唐纳尔·埃伯斯倒是一个热情的人。

第二天负责保护会场安全的唐纳尔·埃伯斯在早上十点开始频繁看表，快到十点半的时候突然对下属说。

「我抽空出去吃午餐，等一下各族政要中午散场我们会更忙。」

五分钟飞车跑回饭店，时间刚好小欣的大餐才端上桌。

池欣他们才坐下，迎面唐纳尔·埃伯斯出现了，嘴里说。

「好巧。」

真的好巧，既然遇到了昨天聊天聊的又那么投机，赶上吃饭还能不让让啊。唐纳尔·埃伯斯也不客气，一屁股就坐下了

。

唐纳尔·埃伯斯吃饭的速度与池欣有一拼，属于风卷残云、让别人看了都觉得饭菜很香的那一类型。而且跟池欣一样吃

得很多，一个人吃三人份，害池欣都不敢放开吃，如果池欣放开了估计他的助手们就要挨饿了。

「你赶时间吗？」

池欣不能容忍有人吃饭的速度和吃下的东西比自己还多。

「恩！中午十二点有工作。」

「你工作的时间和别人不太一样啊？」

「恩，没什么规律的。」

池欣想了一下，他是因为一天要吃四餐所以吃饭时间与别人不同的，这人的作息不会刚好和他吻合吧？

池欣看着唐纳尔·埃伯斯吃饭的样子不得不问。

「冒昧问一下。你多久没吃饭了？」

「从昨天早上和你们分手以后这是第一餐。」

唐纳尔·埃伯斯心里汗颜，工作忙时一个原因，更主要的是，看到饭店提供的餐点就不自觉的想到小欣的好手艺，这样

一对比就更加的食不下咽，所以从昨天早上那顿开始他就开始盼今早这顿了。果然没有让他失望，盘子里最后一块鸡翅

被唐纳尔·埃伯斯从池欣的筷子下面抢了过去。

池欣愣愣看着下筷子比他的速度还快千分之一秒的唐纳尔·埃伯斯，愤恨的收回筷子在桌子上戳齐。现在为止盘盘干净

一点渣也不剩。

唐纳尔·埃伯斯用纸巾擦了嘴似乎才注意到主人盯着他看的眼神。

「对了，昨天玩的愉快吗？」

「很好肖纳·波利尼是个很好的导游。」

「那就好，他的老家就是这里，没有人比他更熟悉这里的人文和景点，有什么需要尽管和他说。」

「谢谢，对了他叫你长官，你是什么官啊？」

「龙族联合事务处安全部司令。」

除了池欣以外，池欣可以知道身边的人似乎有些吃惊，看来蹭饭的家伙真的是大有来头的，不过有什么关系呢，他的身

份也不弱啊，而且对龙族联合事务处的事情池欣本来就不是很关心，那是爷爷和哥哥、姐姐关注的事情。

吃饭吃饭，这个家伙怎么还不走啊，池欣的小点心马上出炉可是没有做他的份啊。

大家心怀鬼胎，最后还是小欣先站起来。池欣实在无法等了，在等一会儿小点心就糊了。

点心上桌的路上池欣先往嘴里塞了三大块，顺便告诉身边的人。

「没吃饱吧，来点点心。」

「公主一边走一边吃不太好吧。」

「端上桌子抢不过那个外国佬，当然要先把自己的份吃出来啊。」

身边的人被池欣说得心动，看看左右没什么人，才犹豫的拿了一块吃。

「公主不要那么小气，现任龙族联合事务处安全部司令别人想请都请不到的。」

「你看到了，我没请他，是他摸着饭点自己来的，明天我们换地方吃饭好了。」

「公主等一下问他明天会不会来，如果他还来作出他那一份就好了。」

「多他一个等于多出三个人的份，明天的菜你去买啊？」

池欣对什么都大方，就是食物方面，连自己都承认有些小气。

「好。我的公主殿下，那个人以后可能用得到的，对他还是客气些好。」

「知道了，比我爸爸还罗嗦。」

对待这个蹭饭的池欣觉得没必要和蔼可亲。

不想多做菜，把吃饭的时间延后一点点都不想，第二天池欣还是决定变更吃饭的地点，他们不在饭厅用餐回房间总可以

吧？

美美吃上一顿，下午在浴池变回龙形泡个澡再舒服不过了，当然不忘记把午餐剩下的小点心放在脖子伸长可以触及的地

方。游上一圈，伸长脖子从盘子里叼上一块吞进去。

「嘿嘿！」

这是池欣感觉最幸福的时刻啊。

可怜这时候的唐纳尔·埃伯斯坐在餐厅里饿得前心贴后背看看时间十一点三十，叫过服务员。

「池欣他们吃过饭了吗？」

「噢唐纳尔·埃伯斯先生，他们今天是在房间里用的餐。」

「噢！天啊！我怎么办？」

服务员要走被唐纳尔·埃伯斯叫住。

「等等，他们一天吃几顿饭，都是自己做的吗？」

「他们一天做四次饭，每次都是自己做，连做菜的材料都是自己出去买的。」

「能不能告诉我他们吃饭的时间？」

「早上十点到十一点，下午两点到三点，晚上六点到七点，夜里十一点到十二点，不过因为我们十一点厨房就要锁门了

，所以他们一般都是十点就开始作了，唐纳尔·埃伯斯您调查他们做什么？是不是他们……」

服务员当唐纳尔·埃伯斯做什么安全调查，池欣他们是什么危险人物了。

「帮我保密，我在进行一个秘密调查。」

唐纳尔·埃伯斯庆幸自己的身份可以为自己调查人家的事情做掩饰。为了蹭饭做的调查让人知道有损威名啊。

当晚夜里十一点，唐纳尔·埃伯斯摸到了小欣门口。敲响了房门，小欣的保镖开了门。

「不好意思今晚有些寂寞，能否进去和你们聊聊天？」

保镖知道他的身份，能说什么，把人让进去。唐纳尔·埃伯斯灵敏的鼻子就闻到了饭菜的香味。

「怎么这么不巧你们在用餐啊？」

「确切的说是池欣一个人在用餐，我们没有吃第四顿的习惯。」

「他在？」

保镖一指浴室里面，晚上的第四顿饭池欣都是变作龙泡在池子里吃的。

唐纳尔·埃伯斯推门进去马上惊呆了，眼前有一只好漂亮的青色小雌龙正愉快的甩着头，岸边一个长案上摆着几样菜肴

。

很健壮，珠圆玉润皮肤散发着诱人光泽的雌性可以说是他平生见过最漂亮的一只，关键是那个身长，根据他的目测怎么

也超过八米了。

唐纳尔·埃伯斯乘这小雌龙正在愉悦的戏水，慢慢靠近食物，虽然美色当前不过食物的诱惑更大一些。他可是饿了一天

多了，水声掩盖了唐纳尔·埃伯斯狼吞虎咽的声响，等小欣注意到的时候饭菜已经少了大半。

有人偷吃了自己的晚餐，池欣迅速的冲回池边眼神愤恨，人家忙碌几十分钟的东西自己还没吃几口呢，怎么就……

池欣把头伸过去瞪着唐纳尔·埃伯斯，唐纳尔·埃伯斯抬头露出牙齿的笑了笑。

「不好意思，刚才进来看你戏水玩的开心，没好打扰，你做的晚餐很好吃，又换菜色了是吗？」

「你是怎么进来的？」

「从大门，你的朋友放我进来的。」

池欣想：看样子这家伙的身份真的很特别了，大半夜自己的保镖会放他进门。

「你来有事吗？」

「一天没看到你们有些想念，过来打个招呼。」

「这么简单？我看你是特意过来蹭饭的。」

小欣小声嘀咕着。

「我想我会爱上中国美食的，要知道我自从尝过你做的菜，就再也吃不下这里的营养餐了。」

唐纳尔·埃伯斯间接的告诉主人，自己自从和他上次分手以后就再也没吃过饭。

「啊！」

池欣一听就知道大事不妙，用嘴巴叼起盘子一仰头把菜倒进了嘴里，这是他的拿手绝活，龙形时真正的狼吞虎咽。

唐纳尔·埃伯斯一看有些傻眼。

「很抱歉我必须吃晚餐，不然晚上会作噩梦的。」

小欣也知道自己的作为有些小气。

当然唐纳尔·埃伯斯也不客气，迅速的变身龙形把小欣的身体挡在了后面，长舌头伸出来一舔就是一盘菜。

「好心的小姐，对不起抢了你的饭，不过我真的饿惨了。」

「你为什么不出去吃饭店？」

小欣真是没遇到过这么厚脸皮的人。

「这个时间饭店已经关门了，大不了我改日请你吃大餐，算是回礼。」

「改天？那今晚怎么办？」

「我相信你还有应急粮食。」

「那又怎么样？我还是想吃饭。」

小欣用自己的头推开唐纳尔·埃伯斯的脖子，两个人似乎都有足够的理由想要独享眼前的晚餐。

推来推去，两人你一口我一口等于分享了剩下的晚餐，吃完饭两人相视而笑，抢饭吃的感觉很好玩。

小欣这才注意看，唐纳尔·埃伯斯龙形是纯黑色的，黑得发亮的那种。

「你有九米了吧？」

「是的，九米三三。」

唐纳尔·埃伯斯的体长远远超过了龙族的平均身长八米。

「你不是人工授精的小孩吧？」

「为什么这么问？」

「没什么啊。」

池欣难得看到几个比自己高大的雄龙，偶然遇到几个都要怀疑人家是不是自己爸爸的种，爷爷说过爸爸年轻的时候去医

错也没犯的，只是没见过面的爷爷，还有样子都要忘记的爸爸站错了边连累我。」

「你这几年怎么过的？」

「做作零工。偶尔偷一些猴子和小菜龙的东西。」

「沧龙族现在的法律还搞株连连坐的吗？长辈的错误为什么连你们都不放过？」

「这你要去问女王，不过要是换作我，我也会这么做，斩草是要除根的。」

「你们沧龙族的作风太过血腥了，也许我可以帮你。」

「帮我？你想从我这里得到什么好处？我连自己的肚子都喂不饱，没什么可以给你的。」

「我帮你不需要好处的，举手之劳，你们沧龙族的公主现在就住在附近不远的饭店里。」

「是不是霸龙饭店。」

「你怎么知道？」

「我偶尔听到有人和我们这边流浪者的老大独眼龙说，只要杀掉霸龙酒店那个小杂种就给他很多钱。」

池漩站起来。

「他们说什么时候动手？」

「不知道，现在沧龙族里有很多的人反对公主做下任女王，说他是沧龙和小菜龙的混血。」

池漩一直等到沧沥吃完饭，和海森打了招呼直奔沧血池住的酒店，多亏他们早来了一步让沧血池有了防备，沧龙族反对

者的刺杀只是让沧血池的胳膊受了轻伤。还要感谢这次的刺杀，沧衃得到消息后大发雷霆，和下面说要提早退位让沧血

池继任女王。任何人休想改变下任沧龙族族长的人选。

沧沥顺利的入了沧龙籍，登记的那一天才知道他是一个雌的。

池漩把他托付给姐姐沧血池照顾，沧血池发现沧沥私藏着池漩的照片。

「你怎么会有他的照片？」

「他是改变我命运的人。王妃真的是个好人。」

「那你就错了，他不是海族三皇子妃，他姐姐才是。」

「啊？」

「他们是双胞胎，他姐姐脾气有些坏，所以有时候就拿他当替身使唤。他是个雄的。」

「啊？」

小沧龙情豆初开，从此恩情转变为暗恋。

沧血池告诉他。

「知道我怎么来的吗？是我妈妈强暴了池族的太子。」

沧沥听后记在心里，他想生一个好脾气雄性沧漩的小龙。

在一个寂静的夜晚，沧沥顺利摸进了池漩的房间。

几经周旋，血飘散在水里，沧沥还是成功了，告别了处子之身强暴了池漩这个好好男人，然后迅速把人打晕跑路了。

池漩费劲周折找到他的时候他已经是带球跑，正是吃什么都反胃的时候。

池漩宣布要和沧沥结婚，第一个反对的就是池璇。

「我不准！你怎么可以娶一只沧龙族的小混混。」

「我喜欢他，而且他现在怀着我的孩子。」

「这是阴谋，你不能上当。」

「姐姐我到了，有自己的生活要过，你不要这样歇斯底里的好不好？」

「哼！你敢娶他我就不认你这个弟弟。」

池漩第一次拒绝了池璇，转身而去。

池继倒是没什么意见，而且砍刀沧沥隆起的肚皮早就笑的合不拢嘴了。

池古告诉儿子。

「你决定了吗？决定了，结婚后就要记住做个大人照顾好老婆孩子。」

「恩！」

池漩是个好丈夫，沧沥怀孕吃不下饭，说很想喝血，沧沥只是说说，他也知道自己嫁了一个素食龙，他那样说会很吓人

，没想到池漩当晚就抽了血给他下饭，只告诉他是血库买来的。

没想到的是池漩的血效果出奇的好，池族的皇血在沧龙嘴里绝对是美味，已经几天吃不下饭的沧沥恢复了食欲。当晚吃

了三大碗饭。

第二天池漩试着从龙族医院买血给沧沥，沧沥竟说没有第一次的好，吃不下。

池漩再一次伸出了胳膊。

第一个发现异状的是池威。他发现池漩脸色越来越差，后来查了一下。

「傻弟弟啊。」

池威找来了脸色发白的池漩。

「对老婆好是没错，你也要考虑自己的身体，你有多少血可以给他下饭的。」

「可他害喜怎么办？我很担心。」

「我已经动员家里人捐赠血浆帮你度过这一个月，除了池湳、池渺弟弟他们还没有成年我没让捐，五弟池淳我都没放过

，你每天去龙族医院找你老婆的主治医生去取吧，据说这股厌食的劲再有半个月左右就会过去了。」

「谢谢大哥。」

「算了亲兄弟不说谢，而且你们也算帮我的忙，自从你家小沧龙过门，爷爷已经很久没有烦我了，我这才有时间做点事

，你下去休息吧，小龙出生你会更累。」

池漩转身手机响起来。一看医院打来的。

「你说什么？最近一次检查结果出来了？双黄卵？」

池漩脚下一个趔趄差点摔到多亏了池威眼疾手快后面扶了一下。

池威安慰他。

「医生说过可能会遗传，别那么紧张，你们不是也很健康的长大了吗？」

「可这是我老婆的第一胎啊？」

本来就没经验偏偏遇到双生子的遗传。

「我去安慰一下沧沥，希望他不要紧张了。」

池威在后面摇头，谁紧张啊？他看来只有他这个弟弟，他跟沧沥谈过的，那个小沧龙说活一天都是赚来的，他本来就是

个混混，这辈子得到了自己最崇拜的人的爱，已经觉得不枉此生了。

顺便说给沧沥办理入籍的时候沧血池发现他不满十八岁，这种情况是要找他监护人签字的，未免麻烦，沧血池作了手脚

硬是给他加了两岁。也就是说沧沥到现在才十七。不过他们谁也没和池漩说怕他有什么心里负担。

「池菁菁。」

池威叫他的助手。

「收拾一下池漩、池璇以前住的套间吧，很快池族就要有两只小龙了，通知爷爷、爸爸、妈妈沧沥肚子里是双胞胎。」

「啊！池漩王子还真能干啊，老族长一定开心死了，我马上就去。」

池威出门看到最小的弟弟池湳从眼前走过不禁露出微笑，池族的皇宫里还真的是小龙不断啊。

——END——
